# SFK Telegram Bot

Неофіційний Telegram бот для зручного перегляду розкладу занять.

> **Disclaimer:** Це неофіційний проект, створений студентом у навчальних цілях. Не пов'язаний з адміністрацією коледжу. Використовує публічне API для отримання даних розкладу.

## Можливості

- ⭐ Швидкий доступ до розкладу
- ⭐ Виправлений неправильний час пар з офіційного розкладу
- ⭐ Відображення наступної пари
- ⭐ Відображення часу до кінця пари
